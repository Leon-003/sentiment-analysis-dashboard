# 📌 Overview

This project presents an **Interactive NLP-Based Sentiment Analysis Dashboard** designed to analyze textual input, detect emotions, and classify sentiment using Natural Language Processing techniques.

The system combines:
- text preprocessing,
- emotion extraction,
- sentiment classification,
- visualization,
- interactive dashboard deployment.

The project uses:
- NLTK,
- VADER Sentiment Analyzer,
- emotion lexicon mapping,
- Streamlit web application.
